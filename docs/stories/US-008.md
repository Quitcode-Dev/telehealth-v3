# US-008: Accessible Portal UI for Low-Digital-Literacy Patients

**Epic:** Not linked
**Priority:** must
**Effort Estimate:** M
**Status:** approved

## User Story

- **As a** elderly or low-digital-literacy patient (Petro)
- **I want** to use the portal with a simplified, accessible interface
- **So that** I can manage my appointments and view results without needing technical assistance

## Acceptance Criteria

- All patient-facing portal screens comply with WCAG 2.1 Level AA accessibility standards, verified by automated scan and manual review
- Minimum body text size across all screens is 14px; all text scales correctly when the device font size is increased to 150%
- No medical abbreviations or clinical jargon appear in patient-facing UI text; all terminology uses plain-language equivalents validated by the Patient Advisory Council
- The portal is fully functional in both Ukrainian and English; the language preference is set at registration and changeable at any time in profile settings
- All interactive elements (buttons, links, form fields) have a minimum touch target size of 44x44px on mobile
- Viber is available as a reminder and notification channel in addition to push notifications and SMS, configurable in patient profile settings
- The portal achieves a System Usability Scale (SUS) score of at least 70 in usability testing conducted with participants aged 55 and above
