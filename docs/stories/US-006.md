# US-006: Family Proxy Account Management

**Epic:** Not linked
**Priority:** must
**Effort Estimate:** XL
**Status:** approved

## User Story

- **As a** parent or proxy user (Sofia)
- **I want** to manage appointments and health records for my children and dependent family members from a single portal login
- **So that** I do not need to call the clinic separately or accompany family members in person for every booking

## Acceptance Criteria

- A registered patient can request to add a dependent profile (child or adult) from their account dashboard
- For child dependants (under 18), the system requires upload of a birth certificate or legal guardianship document before the proxy link is activated
- For adult dependants, the system requires the dependant to provide digital consent via a separate confirmation link sent to the dependant's registered contact
- A clinic administrator can approve or reject proxy access requests within the admin panel; requests must be actioned within 24 hours
- Approved proxy users can book, reschedule, and cancel appointments on behalf of linked dependants
- Proxy users can view lab results for linked dependants only after physician sign-off and only for results dated after the proxy link was activated
- The account dashboard clearly lists all linked dependant profiles with the ability to switch between them in one tap/click
- All proxy access actions are recorded in an audit trail showing the proxy user ID, dependant ID, action taken, and timestamp
