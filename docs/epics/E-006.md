# E-006: Physician & Clinical Staff Portal Interface

**Priority:** must
**Status:** approved

## Description

Provide physicians and clinical staff with a seamlessly integrated interface accessible via SSO from Helsi EHR, delivering pre-visit patient summaries, a shared care team secure messaging inbox, lab result release controls, and digital intake data at the point of care. This epic eliminates the need for physicians to maintain separate portal logins and reduces the 18-minute per-patient administrative overhead.

## Acceptance Criteria

• Physicians access the portal interface via SAML 2.0 SSO from Helsi EHR — no separate login required
• Pre-visit patient summary (reason for visit, current medications, recent labs, allergies, insurance status) is available to the physician before each appointment
• Physicians can approve or release lab results to patient portal with optional contextual note attachment
• Lab result release rules are configurable by result category by each physician
• Shared care team message inbox is accessible within the physician interface and supports asynchronous patient-to-care-team messaging
• Physician interface is validated and signed off by the Physician UX Review Group (2–3 multi-specialty physicians)
• All physician actions (result releases, message responses, note entries) are timestamped and audit-logged
• Interface does not require physicians to leave the Helsi EHR workflow for routine portal tasks
• Physician admin time per patient is tracked as a measurable outcome, targeting reduction from 18 to 8 minutes within 12 months of GA
